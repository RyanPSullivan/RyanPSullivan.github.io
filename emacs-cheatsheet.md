Emacs Cheatsheet
================

* C-x, C-c : end emacs session
* C-g : clear bufferg

Screen Commands
---------------
* C-v : page down
* M-v : page up
* C-l : refresh screen


Cursor Control
-------------
* C-p : previous line
* C-n : next line
* C-b : backward
* C-f : forward character
* C-a : line start
* C-e : line end
* M-b : backward word
* M-f : forward word
* M-a : sentence start
* M-e : sentence end

Misc
----
* C-u : supply a unit (used for repeating commands)
